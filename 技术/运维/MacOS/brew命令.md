
# 1. brew常用命令

brew 搜索软件
```
brew search nginx
```
brew 安装软件
```
brew install nginx
```
brew 卸载软件
```
brew uninstall nginx
```
brew 升级
```
sudo brew update
```
查看安装信息(经常用到, 比如查看安装目录等)
```
sudo brew info nginx
```
查看已经安装的软件
```
brew list
```
# 2. 管理brew安装的服务
```
brew services [-v|--verbose] [list | run | start | stop | restart | cleanup] [...]
```
